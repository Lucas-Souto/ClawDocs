# Blend
```csharp
public enum Blend
```
Define os tipos de blend factor.<br />
|Nome|Valor|Descrição|
|---|---|---|
|Zero|1|0; 0; 0; 0.|
|One|2|1; 1; 1; 1.|
|SourceColor|3|srcR; srcG; srcB; srcA.|
|OneMinusSourceColor|4|1 - srcR; 1 - srcG; 1 - srcB; 1 - srcA.|
|SourceAlpha|5|srcA; srcA; srcA; srcA.|
|OneMinusSourceAlpha|6|1 - srcA;  1 - srcA; 1 - srcA; 1 - srcA.|
|DestinationColor|7|dstR; dstG; dstB; dstA.|
|OneMinusDestinationColor|8|1 - dstR; 1 - dstG; 1 - dstB; 1 - dstA.|
|DestinationAlpha|9|dstA; dstA; dstA; dstA.|
|OneMinusDestinationAlpha|10|1 - dstA; 1 - dstA; 1 - dstA; 1 - dstA.|
